## ‚ÑπÔ∏è Informazioni sul Progetto

Progetto sviluppato come test tecnico per **Dynamic Consult Milano** (23 - 27 Novembre 2025).

**Stack Tecnologico:**
- **Backend**: Java 21, Spring Boot 3.4.12, PostgreSQL 18.1
- **Frontend**: Flutter 3.x, Dart 3.x, Material Design 3

**Caratteristiche Principali:**
- Integrazione 5 API pubbliche NASA (DONKI + CNEOS)
- Sistema dual correlation per eventi spaziali 
- Dashboard interattiva cross-platform 
- Database normalizzato 8 tabelle con 12 indici
- Performance ottimizzate con Caffeine cache e Lazy Loading

**Dataset 2024:** ~4,700 eventi spaziali tracciati e correlati
